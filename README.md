## 🚀 About

Personal portfolio website for **Amarnath Mohandass**, Automation Test Engineer at **Wipro Technologies**, Chennai.

Built with a dark tech aesthetic featuring animated backgrounds, scroll-reveal effects, and a fully responsive layout — designed to impress HR recruiters and hiring managers.

---

## ✨ Features

- 🌑 Dark theme with cyan/green glow effects and animated grid background
- 🖼️ Profile photo with gradient border frame
- 📊 Hero stats — 500+ automated tests, 30% regression reduction
- 💼 Work experience timeline (Wipro · GIB · Ferguson)
- 🛠️ Skills section grouped by category (Automation, Programming, API, DevOps, Management)
- 🗂️ QA Projects showcase (GIB Cards Suite, WS02-Cortex, eCommerce Framework, Mastercard Simulator)
- 🏦 Domain Knowledge section (Banking, FinTech, eCommerce)
- 📬 Contact section with LinkedIn, GitHub, Email, Phone icons
- 🖱️ Custom trailing cursor (desktop only, disabled on mobile)
- 🔁 Scroll-reveal animations with IntersectionObserver
- 📌 Active nav highlight on scroll
- 📅 Dynamic footer year
- 📱 Fully responsive on mobile and tablet
- 🔍 SEO meta tags (description, keywords, author, OpenGraph)
- ⭐ Browser favicon with initials
